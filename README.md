--> student-data-handler-program(Python)
   python Json data automation project

Description:
This is a Python-based automation program that allows users to input student details and process the data using JSON file handling.
The program takes student name and marks as input, stores the data in a JSON file, and then reads the file to filter and display students who scored more than 50 marks.
Prevents data loss by appending new records instead of overwriting the existing records.
---
-> Features:
- Accepts user input (name and marks)
- Stores data in JSON file
- Reads data from JSON file
- Filters students based on marks (>50)
- Displays formatted output
---
-> Technologies Used:
- Python
- JSON
- File Handling
